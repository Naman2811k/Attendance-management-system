📌 Features

✅ 1. Student Management

Add new students

Update existing student details

Delete students

Auto-save data to file



---

✅ 2. Daily Attendance

Mark Present / Absent / Late / Leave

Select date and mark attendance

Prevent duplicate entries for the same date



---

✅ 3. Attendance Reports

View individual student attendance

Calculate overall percentage

Month-wise attendance summary

Display attendance chart (text-based / GUI table)



---

✅ 4. Data Storage

Supports multiple storage files:

students.txt – Student records

attendance.txt – Date-wise attendance

report.txt – Generated reports


(Data saved automatically)


---

✅ 5. GUI (Java Swing)

Uses JTable, JList, JMenuBar, JOptionPane, CardLayout, etc.

Clean, user-friendly interface



---

✅ 6. Extra Features (Advanced)

Search student

Filter attendance by date

Export report to .txt

Attendance percentage calculation

Auto-load data at startup
